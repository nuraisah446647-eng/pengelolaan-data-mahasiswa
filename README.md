# pengelolaan-data-mahasiswa
Aplikasi desktop berbasis Java Swing untuk pengelolaan data mahasiswa, meliputi fitur tambah, ubah, hapus, pencarian data, serta perhitungan IPK yang terintegrasi dengan database MySQL. Aplikasi ini dibuat sebagai tugas mata kuliah Aplikasi Mobile 1.

## Analisis Masalah
Pengelolaan data mahasiswa secara manual sering menimbulkan masalah seperti
kesalahan pencatatan, data sulit dicari, dan proses pengolahan nilai yang
memakan waktu. Oleh karena itu, diperlukan aplikasi yang dapat membantu
pengguna dalam mengelola data mahasiswa secara cepat dan akurat.

## Pengguna Aplikasi
Aplikasi ini ditujukan untuk:
- Dosen
yang bertugas mengelola data mahasiswa dan nilai perkuliahan.

## Alur Kerja Aplikasi
1. Pengguna melakukan login ke aplikasi
2. Pengguna menambahkan atau mengelola data mahasiswa
3. Sistem menyimpan data ke database
4. Nilai IPK dihitung secara otomatis
5. Data ditampilkan dalam bentuk tabel
6. Pengguna dapat mencari, mengedit, atau menghapus data
